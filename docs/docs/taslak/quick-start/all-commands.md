# Summary
Arguments usage defination for `foo bar` command
* exactly once 'name' arg: `pars foo bar name`
* exactly 3 'name' arg: `pars foo bar name1 name2 name3`
* optional 'name' arg: `pars foo bar [name]`
* at least 1 'name' arg: `pars foo bar name1 [name2]...`
* at least 2 'name' arg: `pars foo bar name1 name2 [name3]...`
* at most 1 'name' arg: `pars foo bar [name]`
* at most 3 'name' arg: `pars foo bar name1 name2 [name3]`
* unlimited 'name' arg: `pars foo bar name1 [name2] [name3]...`
