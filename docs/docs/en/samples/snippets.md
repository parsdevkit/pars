## Inline Format
### Basic
--8<-- ".browserslistrc"


### Single line
--8<-- "nginx-sample.log:4:4"

### Line range
--8<-- "nginx-sample.log:4:6"

### From Section
--8<-- "samples/admonitions.md:adding-title"

.browserslistrc
nginx-sample.log
